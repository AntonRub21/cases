A Telegram App Center-style CS2 cases web app with:

- Case and skin **images** (not just text).
- Real case opening reel animation with scrolling skin cards.
- Single in-app currency: **Coins**.
- User inventory with image cards and Steam withdrawal requests.
- Separate admin page for case management, coin balance updates, and withdrawal moderation.
- JSON-based user database persisted in `app.db.json`.

## Run

```bash
npm start
```

Open:
- User app: `http://localhost:3000/`
- Admin app: `http://localhost:3000/admin.html`

## Test

```bash
npm test
```
